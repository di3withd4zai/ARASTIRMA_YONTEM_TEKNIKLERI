# MP3 – Veri Toplama ve Hazırlık

Bu aşamada araştırma kapsamında kullanılacak olan anket verileri Google Forms aracılığıyla toplanmıştır. Toplam 253 katılımcıdan elde edilen veriler CSV formatına dönüştürülmüş ve Python ortamında analiz edilebilmesi için düzenlenmiştir.

Bu klasörde:
- `data_mp3.csv` dosyası (ham veri)
- Veri sütunlarının açıklamaları
- Projede kullanılacak analiz adımlarının temel taslağı

yer almaktadır.

MP3 aşaması, MP4 veri analizi sürecinin temelini oluşturur.
