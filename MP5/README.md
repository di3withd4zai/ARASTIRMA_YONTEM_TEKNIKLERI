# MP5 – IEEE Formatında Akademik Rapor

Bu klasör, proje kapsamında üretilen tüm analizlerin akademik formatta derlendiği IEEE Conference Template uyumlu raporu içermektedir. Rapor; problem tanımı, veri toplama süreci, istatistiksel analizler, grafikler, bulgular ve kaynakçayı içeren kapsamlı bir çalışmadır.

Bu klasörde:
- `MP5_Report.pdf` (IEEE formatında final rapor)
- `MP5_Report.tex` (Overleaf/LaTeX dosyası)
- Analiz görselleri (heatmap, histogram, boxplot vb.)

yer almaktadır.

MP5 aşaması, projenin akademik çıktı olarak sunulduğu son aşamadır.
